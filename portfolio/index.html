1. WHP

▶ GitHub

3. TokenHunter

▶ Unity   ▶ GitHub

챗봇 모델을 이용한 채팅 형식의 추리 소설 게임입니다.

ChatGPT API, 음성 합성(TTS) 및 음성 인식(STT) 기능을 활용했습니다.
플레이어가 심문관이 되고, AI가 용의자가 되어 진행되는 추리 소설 게임입니다.
제한된 토큰 내에서 플레이어는 용의자인 챗봇으로부터 범행 사실을 자백하도록 유도해야 합니다.

Scene 1명, Animation 1명, DevOps 1명

2024. 10 ~ 2024. 12
